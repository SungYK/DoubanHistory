# INTRO

这是一个收集院线上映电影的豆瓣历史评分的网站，源于突然有一天想了解一部电影豆瓣评分变化，搜索之后发现还没有这样的网站， 也确实有人手动截图在记录关心的影片/剧集的评分变化，特别是在讨论关于水军/刷分等话题时，于是就自己写了一个。 由于是第一次做，没什么经验，目前还是比较简陋，更多东西会慢慢完善。

如果对你有所帮助，希望你可以分享给其他人。

  • 目前只记录展示了正在上映的影片评分变化，记录的时间范围为上映至此影片不再排片为止
  • 已下映影片可通过搜索查询（数据从2018年2月开始收集）
  • 数据来源于 豆瓣 API
  • 前端图表展示 Echarts
